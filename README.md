# HTML Authentication POC

This project contains 5 HTML authentication pages created using only HTML tables and anchor tag redirections.

Pages Included:

1. login.html
2. register.html
3. forgot-password.html
4. reset-password.html
5. dashboard.html

All redirections are handled using anchor tags only.
